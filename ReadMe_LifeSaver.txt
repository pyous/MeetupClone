 ======= TO GET THE APPLICATION ON AN ANDROID EMULATOR ========
1: Modify the default DSK on Genymotion (After installing Android Studio)
2: Run : npm install -g exp
3: Run : exp path
4: Add ADB to your environnement PATH (On Windows)
5: Kill ADB server
6: Close EXP and Genymotion
7: Open them, and enjoy :)
